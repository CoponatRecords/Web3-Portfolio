### 1. Clone the Repository

```bash

```

### 2. Install Dependencies

```bash
npm install

```

### 3. Create .env.local

Set up your environment variables in a .env.local file:

```bash
NEXT_PUBLIC_ALCHEMY_API_KEY=your_alchemy_key
NEXT_PUBLIC_0X_API_BASE=https://api.0x.org
```

### 4. Run the App

```bash
npm run dev
# or
yarn dev
```
